bellman.trajectory_optimisers.cross_entropy_method.cross_entropy_method_trajectory_optimisation¶

bellman.trajectory_optimisers.cross_entropy_method.cross_entropy_method_trajectory_optimisation(time_step_spec, action_spec, horizon, population_size, number_of_particles, num_elites, learning_rate, max_iterations)[source]¶

Helper function to construct a trajectory optimiser which uses the cross entropy method.

Parameters

  • time_step_spec (TimeStep) – A TimeStep spec of the expected time_steps.

  • action_spec (Union[ForwardRef, Iterable[ForwardRef], Mapping[str, ForwardRef]]) – A nest of BoundedTensorSpec representing the actions.

  • horizon (int) – Number of steps taken in the environment in each virtual rollout.

  • population_size (int) – The number of candidate sequences of actions at each iteration.

  • number_of_particles (int) – Number of monte-carlo rollouts of each action trajectory.

  • num_elites (int) – The number of elite trajectories used for updating the parameters of distribution for each action. This should be a proportion of population_size rollouts.

  • learning_rate (float) – The learning rate for updating the distribution parameters.

  • max_iterations (int) – The maximum number of iterations to use for optimisation.

Return type

TrajectoryOptimiser

Returns

A TrajectoryOptimiser object which uses the cross entropy method.

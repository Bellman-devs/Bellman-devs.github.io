bellman.trajectory_optimisers.particles.reshape_create_particle_axis¶

bellman.trajectory_optimisers.particles.reshape_create_particle_axis(batched_particles_tensor, number_of_particles)[source]¶

Given a tensor with a single batch dimension and shape [population_size * number_of_particles, …], return a reshaped tensor with two batch dimensions and shape [population_size, number_of_particles, …].

Parameters

  • batched_particles_tensor (Union[Tensor, SparseTensor, RaggedTensor]) – A Tensor with shaped [batch_size, …].

  • number_of_particles (int) – An integer corresponding to the number of duplicates of the initial batched tensor.

Return type

Union[Tensor, SparseTensor, RaggedTensor]

Returns

A Tensor with shape [reduced_batch_size, multiplier, …].

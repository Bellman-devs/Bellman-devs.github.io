bellman.trajectory_optimisers.trajectory_optimisers.TrajectoryOptimiser¶

class bellman.trajectory_optimisers.trajectory_optimisers.TrajectoryOptimiser(batch_size)[source]¶

Bases: abc.ABC

A trajectory optimiser optimises a sequence of actions given a model of the the environment that is used for virtual rollouts.

Initializes the class.

Parameters

batch_size (int) – The number of virtual rollouts that are simulated in each iteration during optimization.

abstract optimise(time_step, environment_model)[source]¶

Optimise trajectories starting from an initial time_step.

Parameters

  • time_step (TimeStep) – Initial TimeStep from which rollouts are starting.

  • environment_model (EnvironmentModel) – An EnvironmentModel is a model of the MDP that represents the environment, consisting of a transition, reward, termination and initial state distribution model, of which some are trainable and some are fixed.

Returns

Action trajectory (Tensor with shape (horizon,) + action_shape).

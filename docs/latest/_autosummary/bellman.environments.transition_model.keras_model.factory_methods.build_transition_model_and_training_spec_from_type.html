bellman.environments.transition_model.keras_model.factory_methods.build_transition_model_and_training_spec_from_type¶

bellman.environments.transition_model.keras_model.factory_methods.build_transition_model_and_training_spec_from_type(observation_spec, action_spec, transition_model_type, num_hidden_layers, num_hidden_nodes, activation_function, ensemble_size, predict_state_difference, epochs, training_batch_size, callbacks, trajectory_sampler, observation_transformation=None, verbose=0)[source]¶

Custom function to build a keras transition model plus training spec from arguments.

Parameters

  • observation_spec (Union[TypeSpec, TensorSpec, RaggedTensorSpec, SparseTensorSpec, ForwardRef, Iterable[ForwardRef], Mapping[str, ForwardRef]]) – A nest of BoundedTensorSpec representing the observations.

  • action_spec (Union[TypeSpec, TensorSpec, RaggedTensorSpec, SparseTensorSpec, ForwardRef, Iterable[ForwardRef], Mapping[str, ForwardRef]]) – A nest of BoundedTensorSpec representing the actions.

  • transition_model_type (TransitionModelType) – An indicator which of the available transition models should be used - list can be found in TransitionModelType. A component of the environment model that describes the transition dynamics.

  • num_hidden_layers (int) – A transition model parameter, used for constructing a neural network. A number of hidden layers in the neural network.

  • num_hidden_nodes (int) – A transition model parameter, used for constructing a neural network. A number of nodes in each hidden layer. Parameter is shared across all layers.

  • activation_function (Callable) – A transition model parameter, used for constructing a neural network. An activation function of the hidden nodes.

  • ensemble_size (int) – A transition model parameter, used for constructing a neural network. The number of networks in the ensemble.

  • predict_state_difference (bool) – A transition model parameter, used for constructing a neural network. A boolean indicating whether transition model will be predicting a difference between current and a next state or the next state directly.

  • epochs (int) – A transition model parameter, used by Keras fit method. A number of epochs used for training the neural network.

  • training_batch_size (int) – A transition model parameter, used by Keras fit method. A batch size used for training the neural network.

  • callbacks (List[Callback]) – A transition model parameter, used by Keras fit method. A list of Keras callbacks used for training the neural network.

  • trajectory_sampler (TrajectorySamplingStrategy) – Trajectory sampler determines how predictions from an ensemble of neural networks that model the transition dynamics are sampled. Works only with ensemble type of transition models.

  • observation_transformation (Optional[ObservationTransformation]) – To transform observations to latent observations that are used by the transition model, and back. None will internally create an identity transform.

  • verbose (int) – A transition model parameter, used by Keras fit method. A level of how detailed the output to the console/logger is during the training.

Return type

Tuple[KerasTransitionModel, KerasTrainingSpec]

Returns

The keras transition model object and the corresponding training spec.

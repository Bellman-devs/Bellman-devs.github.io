bellman.harness.utils.get_tag_names¶

bellman.harness.utils.get_tag_names(root_dir, experiment_phase, experiment_dir)[source]¶

When running an experiment through the experiment harness, tensorboard event files are created with metrics recorded during the experiment. Each metric gets a unique tag name that has to be used to retrieve metric values with get_metric_values. Tensorflow generates a lots of metrics that are recorded and this function can be used to get tag names associated with each metric.

Parameters

  • root_dir (str) – The root directory used by the experiment harness.

  • experiment_phase (str) – The phase of the experiment in which the trace was recorded.

  • experiment_dir (str) – An experiment id which has been recorded in the root directory.

Return type

Set[str]

Returns

A set of tag names.

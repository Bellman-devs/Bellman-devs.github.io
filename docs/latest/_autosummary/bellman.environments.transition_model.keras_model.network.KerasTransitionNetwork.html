bellman.environments.transition_model.keras_model.network.KerasTransitionNetwork¶

class bellman.environments.transition_model.keras_model.network.KerasTransitionNetwork(bootstrap_data=False)[source]¶

Bases: abc.ABC

This class defines the structure and essential methods for a transition network. The transition network is a sequential, feed forward neural network. It also makes it easy to create networks where data is bootstrapped for each new network in an ensemble.

Subclasses of this class should define the structure of the network by implementing the build_model method. The output layer should be reshaped to match the observation tensors from the environment. The loss function of the network should be specified by implementing the loss method. The training data can be manipulated by overriding the transform_training_data method with the appropriate transformation.

Parameters

bootstrap_data (bool) – Create an ensemble version of the network where data is resampled with replacement.

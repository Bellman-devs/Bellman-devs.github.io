bellman.environments.utils.create_real_tf_environment¶

bellman.environments.utils.create_real_tf_environment(env_name, random_seed=None, spec_dtype_map=None, gym_kwargs=None)[source]¶

This function creates a gym environment object by loading the spec by name from gym registry, and wrapping it in a TFPyEnvironment for compatibility with TF-Agents.

Parameters

  • env_name (str) – Name for the environment to load.

  • random_seed (Optional[int]) – Value to use as seed for the environment.

  • spec_dtype_map (Optional[Dict[Space, dtype]]) – Maps gym spaces to np dtypes to use as the default dtype for the arrays.

  • gym_kwargs (Optional[Dict[str, Any]]) – Optional kwargs to pass to the Gym environment class.

Return type

TFEnvironment

Returns

A TFEnvironment which wraps the real gym environment.

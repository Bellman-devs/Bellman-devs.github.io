bellman.agents.trpo.trpo_agent.compute_return_and_advantage¶

bellman.agents.trpo.trpo_agent.compute_return_and_advantage(discount_factor, lambda_, rewards, next_time_steps, value_preds)[source]¶

Compute the TD-lambda return and GAE(lambda) advantages. Normalization will be applied to the advantages.

Parameters

  • discount_factor – discount in [0,1]

  • lambda – trace_decay in [0,1]

  • rewards – next_step rewards (possibly normalized)

  • next_time_steps – batched tensor of TimeStep tuples after action is taken.

  • value_preds – Batched value prediction tensor. Should have one more entry in time index than time_steps, with the final value corresponding to the value prediction of the final state.

Returns

tuple of (return, normalized_advantage), both are batched tensors.
